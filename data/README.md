## Spam Detection Dataset

Download the dataset from Google Drive:

- Link: https://drive.google.com/file/d/1r_PT8dqvQe-gjEeccF7mgED3a98zZmXV/view

Quick download (requires gdown):
```bash
pip install gdown
gdown https://drive.google.com/uc?id=1r_PT8dqvQe-gjEeccF7mgED3a98zZmXV -O data/spam-dataset.zip
unzip data/spam-dataset.zip -d data/
```
